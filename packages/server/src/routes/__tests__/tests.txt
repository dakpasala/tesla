Test the DB:
    Get parking availability: curl "http://localhost:3000/api/parkings/?loc_name=Palo%20Alto%20Office"
    Update parking availability: curl -X PATCH http://localhost:3000/api/parkings/ \
        -H "Content-Type: application/json" \
        -d '{
            "loc_name": "Fremont Factory",
            "lot_name": "North Lot",
            "availability": 10
        }'

    Get admins: curl http://localhost:3000/api/admins/
    Add admin: curl -X POST http://localhost:3000/api/admins/ \
    -H "Content-Type: application/json" \
    -d '{"username":"user","email":"user@calpoly.edu"}'

    Add to a user's incentives: curl -X POST http://localhost:3000/api/users/1/incentives \
        -H "Content-Type: application/json" \
        -d '{ "transitType": "shuttle" }'
    Get a user's balance: curl http://localhost:3000/api/users/1/balance
    Get a user's incentives: curl http://localhost:3000/api/users/1/incentives
    Get home address: curl http://localhost:3000/api/users/1/home_address
    Update home address: curl -X PUT http://localhost:3000/api/users/1/home_address \
        -H "Content-Type: application/json" \
        -d '{"homeAddress":"789 California Blvd, Palo Alto, CA"}'
    Get work address: curl http://localhost:3000/api/users/1/work_address
